## 💪 نقاط القوة الفريدة لمحسوب (الميزة التنافسية)

### 1. 🤖 **AI-First** — لا يوجد منافس
**محسوب:** اكتب "فاتورة لأحمد 5000 ريال تصميم موقع" → فاتورة جاهزة
**Vom + Qoyod:** املأ 8 حقول يدوياً

### 2. 🗣️ **لغة بسيطة لغير المحاسبين**
**محسوب:** "الفلوس عند الناس" بدل "الذمم المدينة"
**Vom + Qoyod:** "ميزان مراجعة، دفتر أستاذ، قيود يومية"

### 3. 📥 **رفع إغلاقات الكاشير بالذكاء**
**محسوب:** ارفع PDF/صورة → النظام يستخرج كل شي
**Vom + Qoyod:** إدخال يدوي

### 4. 💬 **مستشار مالي 24/7**
**محسوب:** اسأل عن مشروعك بالعربي
**Vom + Qoyod:** غير موجود

### 5. ⚡ **بداية في 5 دقايق**
**محسوب:** اسم + إيميل + كلمة سر → داخل
**Vom + Qoyod:** Onboarding طويل + تدريب

### 6. 🎨 **تصميم عصري + هوية بصرية مميزة**
**محسوب:** ألوان دافئة، خطوط حديثة، تجربة Notion-like
**Vom + Qoyod:** واجهات تقليدية محاسبية

## 🎲 الاستراتيجية المقترحة لمحسوب

### المرحلة 1: التموقع (الآن - 3 شهور)
**🎯 الجمهور:** أصحاب المشاريع الصغيرة جداً (مبيعات < 100K/شهر) الذين:
- لا يفهمون المحاسبة
- لا يستطيعون توظيف محاسب
- يريدون شيئاً بسيطاً وذكياً

**❌ تجنّب:** المنافسة على الشركات المتوسطة والكبيرة (هذي ساحة Vom + Qoyod)

**📣 الرسالة التسويقية:**
> "محاسبتك صارت سهلة. اكتب فاتورتك بالعربي، ومحسوب يتولّى الباقي."

**💰 التسعير المقترح:**
- **مجاني:** 14 يوم تجربة + بعدها 5 فواتير/شهر مجاناً للأبد
- **أساسي:** 49 ر.س/شهر — للفريلانسرز (فواتير غير محدودة + AI)
- **النمو:** 99 ر.س/شهر — للمشاريع الصغيرة (+ موظفين + إغلاقات يومية)
- **الاحترافي:** 199 ر.س/شهر — للمتاجر (+ مخزون + موردين + متعدد فروع)

> 💡 السعر **أقل بنسبة 40-70%** من Vom (270 ر.س) و50% من Qoyod (200 ر.س)

### المرحلة 2: سدّ الفجوات (3-9 شهور)
بالترتيب:
1. ✅ إدارة المصاريف والموردين (شهر 1)
2. ✅ سندات قبض/صرف (شهر 1)
3. ✅ مخزون متقدم (شهر 2)
4. ✅ تسوية بنكية بسيطة (شهر 3)
5. ✅ رواتب مبسّطة (شهر 4)
6. ✅ تطبيق جوال (شهر 6)

### المرحلة 3: التوسع (9 شهور+)
- API عام
- تكاملات مع POS (Foodics, Zid)
- متعدد الفروع
- متعدد العملات
